# Roguelike经营类游戏《拉郎配》

《SHIPPING（拉郎配）》是一款融合了策略与经营元素的Roguelike游戏。玩家将扮演一名贪婪的婚介师，经营一家婚姻介绍所，每天和形形色色的单身人士打交道。玩家可以通过自己的能力润色候选人简历，强行满足客户的需求，赚取佣金。玩家的最终目标是在世界各地建设分店，将公司建设成最强连锁婚介所。
